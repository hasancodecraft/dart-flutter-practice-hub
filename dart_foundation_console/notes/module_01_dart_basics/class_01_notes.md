## Fixed Digits After Decimal Point

**Method:** `toStringAsFixed()`

`toStringAsFixed(2)` is used to show 2 digits after the decimal point.

**Example:**

double cg = 3.5;
String formattedCg = cg.toStringAsFixed(2);

print(formattedCg);
